# 프론트엔드 디자인 언어

목표: **"AI가 만든 티가 안 나는", 조용하고 밀도 있는 도구.** Linear·Arc·Bear 같은 "quiet tool" 계열의 절제 + 한국어 타이포그래피 중심. 화려함보다 매일 열어도 질리지 않는 쪽.

## 1. 하지 않을 것 (AI 티 목록)

- 보라→파랑 그라데이션, glassmorphism, 네온 글로우
- 카드마다 큰 drop-shadow, 과한 border-radius(20px+)
- 이모지 남발, "✨ AI가 요약했어요!" 류의 문구 — AI는 결과로만 존재하고 UI에서 떠들지 않는다
- 히어로 섹션 + 3D 일러스트 랜딩 — 이 제품은 로그인하면 바로 서랍이 열리는 도구다
- shadcn 기본 스킨 그대로 쓰기 (컴포넌트 로직만 차용, 색·radius·간격은 전부 자체 토큰)

## 2. 디자인 토큰

**타이포**: Pretendard Variable (본문/UI 전부). URL·도메인·타임스탬프 등 메타데이터만 모노스페이스(JetBrains Mono 또는 시스템 mono, 11~12px)로 — 이 대비가 "도구" 느낌을 만드는 핵심 디테일.

**컬러 — A안 "먹과 주홍" (✔ 2026-07-22 확정)**:

| 토큰 | 값 | 용도 |
| --- | --- | --- |
| bg | `#FAF9F7` | 앱 배경 (웜 화이트, 종이 느낌) |
| surface | `#FFFFFF` | 카드 |
| ink | `#1C1B1A` | 본문 텍스트 |
| ink-muted | `#6E6A64` | 보조 텍스트, 메타 |
| line | `#E7E4DF` | 보더 (그림자 대신 1px 보더가 기본) |
| accent | `#D4520E` | 주홍(단청/감 계열) — 링크 hover, 핀, 활성 필터, 포커스 링 |
| accent-soft | `#FBEDE4` | 액센트 배경 (활성 태그 칩 등) |

**B안 "먹과 심록"**: accent `#1F6A4E`, accent-soft `#E6F1EB`, 나머지 동일. (차분한 서재 느낌)

원칙: **액센트는 한 색만, 전체 화면의 5% 미만.** 상태색은 성공 없이 실패(`#B3261E`)와 진행(ink-muted pulse)만.

radius: 6px 통일(칩만 full). 그림자: 오버레이(모달/팝오버)에만 허용.

## 3. 레이아웃과 시그니처 인터랙션

```
┌────────────────────────────────────────────────┐
│  ⌘K  [ 검색하거나, URL을 붙여넣으세요        ] │  ← 옴니바 (상단 고정)
├──────────┬─────────────────────────────────────┤
│ 사이드바  │  카드 그리드 (균일 3~4열, 반응형)     │
│ 전체      │  ┌─────┐ ┌─────┐ ┌─────┐           │
│ 📌 고정   │  │thumb│ │thumb│ │ ⋯분석중│         │
│ ─태그─    │  │제목  │ │제목  │ │도메인 │          │
│ #개발 12  │  │요약2줄│ │요약2줄│ └─────┘         │
│ #요리 5   │  │#태그 칩│ …                        │
│ (색 점)   │  └─────┘                            │
└──────────┴─────────────────────────────────────┘
```

**시그니처: 옴니바 하나로 검색과 저장을 겸한다.** 입력값이 URL이면 "저장" 모드(Enter → 카드 생성), 아니면 즉시 검색 필터링. 입력창 오른쪽 끝에 현재 모드가 작게 표시(`저장 ↵` / `검색`). 별도 "추가 버튼 + 모달" 없음 — 이 단일 진입점이 제품의 특색.

카드 상세:

- 썸네일 16:9 고정(없으면 favicon + 도메인 이니셜의 단색 플레이스홀더 — 랜덤 그라데이션 금지)
- 제목 최대 2줄, 요약 최대 2줄, 하단에 파비콘 + 도메인(mono) + 상대 시간
- 태그 칩: 해시(#) 없이 소형 필, 태그 색은 4px 점으로만 표현 (칩 전체를 칠하지 않음)
- **분석 중 카드**: 스피너 대신 도메인 + 미세하게 pulse하는 점 하나 + 스켈레톤 2줄. 완료 시 fade-in 교체 (150ms, 과한 스프링 애니메이션 금지)
- 실패 카드: 좌측 3px 빨간 보더 + failReason 한 줄 + [재시도] 텍스트 버튼

hover: 카드 보더가 ink로 진해지는 정도 (lift/scale 금지). 포커스: accent 2px 링 (키보드 내비 완전 지원).

## 4. 화면 목록 (MVP)

1. `/login` — 로고 + 구글 로그인 버튼 하나. 그게 전부인 화면.
2. `/` 대시보드 — 위 레이아웃. 빈 상태: "첫 링크를 붙여넣어 보세요" + 옴니바로 포커스 이동.
3. 카드 클릭 → 우측 슬라이드 패널(모달 아님): 큰 썸네일, 요약/메모 인라인 편집, 태그 편집, 원문 열기, 삭제.
4. `/settings` — 태그 관리(rename/색/병합), 계정.

다크 모드는 Phase 5 (토큰이 이미 시맨틱이라 값만 갈아끼우면 됨).

## 5. 문구 톤

- 한국어, 해요체 최소화 — 레이블은 명사형("저장", "재시도", "고정됨").
- 에러는 원인 + 다음 행동: "본문을 가져오지 못했어요 · 재시도"
- AI 언급은 설정 화면 외에는 하지 않는다.
